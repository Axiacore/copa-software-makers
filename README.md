### Inscripción

Para registrar tu equipo debes crear un [Pull request]() donde listes el nombre de tu equipo y los miembros en  **[Equipos.md]()**. Ejemplo:


**Nombre** Acme
**Empresa**: Acme Inc.
**Integrantes**:

* John Doe - @johndoe
* Jeny Doe - @jennydoe
* Shaina Shupp - @sshup
* Brenton Friedel @brfrie
* Pepito perez - @pperez

Cada equipo debe contar con 5 jugadores
